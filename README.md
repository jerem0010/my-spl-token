# my-spl-token

A small Solana program built with Anchor to experiment with the basics of SPL tokens.

## What it does

* initializes a new SPL token mint
* sets mint and freeze authority
* mints tokens to a token account
* burns tokens from a token account

## Tech

* Rust
* Solana
* Anchor
* anchor-spl
